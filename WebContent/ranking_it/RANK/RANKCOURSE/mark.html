Spiegazione icone.

Le seguenti icone appariranno accanto al nome del giocatore nella classifica.

Queste icone rappresentano la condizione dell'auto in una corsa specifica. Nell'esempio soprastante, le impostazioni dell'auto sono state modificate, è stata utilizzata la funzione di assistenza ed il tempo migliore si è ottenuto in Modalità Guida.

Quest'icona indica che le impostazioni dell'auto sono state modificate.

Quest'icona verrà visualizzata quando si utilizza una funzione di assistenza in una corsa. Anche se nel giro migliore non ci si è serviti di tale funzione, si terrà conto delle altre occasioni in cui è stata utilizzata.

Queste icone rappresentano le 3 modalità: Pratica, Guida,Gara (da sinistra a destra). Nella modalita Gara, utilizzando il risucchio abilmente, potrai effettuare giri più veloci rispetto ad una corsa a singolo giocatore.

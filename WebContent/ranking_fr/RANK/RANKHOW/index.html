Comment s'inscrire sur Internet Ranking.


Veuillez suivre les instructions suivantes pour vous inscrire.

1. Sauvegardez les données de votre jeu sur votre VM.
Après avoir joué "F355 Challenge", sélectionnez "Save Game" sur le Main Menu et sauvegardez les données de votre jeu dans VM



2. Enregistrez vos données personnelles.
Si vous vous inscrivez pour la première fois, vous devez d'abord enregistrer vos données personnelles. Insérez un VM dans votre Dreamcast et allez dans Player Data Screen. Cliquez sur "Personal Data Registration Form". Compléter le formulaire et inscrivez-vous.

Allez dans Player Data Screen ---> GO



3. Inscrivez-vous sur l'Internet Ranking.
Insérez un VM dans votre Dreamcast, et allez dans Player Data Screen. Cliquez sur le bouton "Ranking Registration".

Allez dans Player Data Screen ---> GO



Suivez les instructions ci-dessus pour enregistrer vos données.

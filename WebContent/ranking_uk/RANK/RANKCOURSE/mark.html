Explanation on the icons

You will see the following icons displayed next to the name of the player on a ranking table.

These icons represent the condition of the car for that specific run. In the example above, the car settings have been modified, an assist function was used, and the best time was recorded in Driving Mode.

This icon shows that the car settings have been modified.

This icon will be displayed when an assist function was used in the run. Even if no assist functions were used in the best lap, if they had been used in the previous or the following laps, it will be considered to have been used.

These icons represent the 3 modes - Training, Driving and Race (in order from the left). In Race mode, by using slip streams skillfully, you can get faster laps compared to a solo run.

Come registrarsi alla Classifica Internet.


Segui le istruzioni sottostanti per registrarti.

1. Salva i dati di gioco sulla VM.
Dopo aver giocato a "F355 Challenge", seleziona Salva Gioco dal menu principale e salva i dati di gioco sulla VM.



2. Registra i tuoi dati personali.
Se ti stai registrando per la prima volta, prima devi registrare i tuoi dati personali. Inserisci la VM nel Dreamcast, vai alla schermata dei dati del giocatore e clicca sul tasto Modulo di Registrazione Dati Personali, riempi il modulo e registrati.

Vai allo schermo Dati del Giocatore. ---> GO



3. Registrati nella Classifica Internet.
Inserisci la VM nel tuo Dreamcast e vai allo schermo Dati del Giocatore. Clicca sul tasto Registrazione a Classifica.

Vai allo schermo Dati del Giocatore. ---> GO
